# Stable Back End

## Development

Create a `.env` file:

```sh
cp .env.example .env
```

Install dependencies:

```sh
yarn install
```

Run the development server by default on [http://localhost:3001]():

```sh
yarn start:dev
```

Swagger UI is available at [http://localhost:3001/api](), and in JSON format at [http://localhost:3001/api-json]().
